Git-cloc
====
Count lines of code for GIT


Examples
====

Count lines of a specified git repo

    git cloc

Search all git repos within the current directory
    
    git cloc -r
    
Show git-cloc manual
    
    git cloc --help
    
    # or
    git help cloc

Install
====
    
    sudo sh install.sh
